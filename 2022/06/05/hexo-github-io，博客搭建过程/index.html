前言

这是个人博客的第一篇文章，简单记录了博客网站的搭建过程。

本博客是基于Hexo 和GitHub Pages搭建的。

  • Hexo是基于Node的开源博客框架，支持markdown格式，支持一键部署各平台；
  • Github Pages是Github提供的免费（重点）静态站点服务，配置OK后，可以通过[你的GitHub用户名].github.io域名访问。

一、安装hexo

安装Hexo很简单，但需要先安装Node.js和Git。

  1. 安装Node.js：

    • 下载并安装Node.js。

    • 安装完成后，打开一个终端，输入node -v，如果返回node版本号，则说明安装成功。

  2. 安装Git：

    • 下载并安装 git；
    • 完成安装后，输入git --version，如果返回版本号，则说明安装成功；
